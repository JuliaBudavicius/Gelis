Sobre

Somos uma empresa o de monitoramento de Hardware e temperatura de computadores. Nosso foco são vocês, empresas de renderização e edição de videos. A GELI'S é a empresa que vai fazer a diferença para você e para sua empresa. Vamos trabalhar juntos!

A equipe do GELI'S é composta por 5 integrantes. Todos os integrantes são alunos de Analise e Desenvolvimento de Sistemas na Faculdade bandTec.

Todos trabalham diaramente em melhorias para que a HORUS entregue a melhor funcionalidade para os seus clientes.

Horus é a nossa aplicação "mãe", através dela que nós realizamos os nossos monitoramentos e, tomamos medidas necessarias para ajudar vocês nossos clientes.

O Slack é a plataforma que o nosso software utilizará para avisar sobre o status da temperatura e do hardware do computador.

Nossa aplicação gera uma Dashboard que através dela o usuario tera um controle de todos os computadores com a nossa aplicação. Através dela o ususario tem um controle de qual computador esta com configurações "Ideias" ou "Não adquadas".
